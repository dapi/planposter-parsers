парсер сайта http://theoryandpractice.ru
на вход можно подсунуть 2 параметра

include_snapshot - включить в выходный файл json снимок страницы на момент разбора.
debug_mode - включить вывод отладочной информации, записываемой в файл debug.log

на выходе парсер генерирует файлы (1 файл на 1 событие) в папке data.

парсер проверялся на конфигурации WAMP.
